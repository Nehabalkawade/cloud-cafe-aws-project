# ☕ Cloud Cafe AWS Project

## 📌 Project Overview

Cloud Cafe is a dynamic web application deployed on AWS using an Application Load Balancer (ALB) and multiple EC2 instances.

This project demonstrates load balancing, high availability, and dynamic content delivery using Node.js and AWS services.

---

## 🚀 AWS Services Used

* Amazon EC2
* Application Load Balancer (ALB)
* Target Groups
* Security Groups
* AWS VPC
* Node.js
* PM2

---

## ✨ Features

* Dynamic menu display
* Auto refresh every 2 seconds
* Load balancing across multiple EC2 instances
* Server identification (Server 1 / Server 2)
* High availability architecture
* Responsive web interface

---

## 🏗️ Architecture

User Browser
↓
Application Load Balancer (ALB)
↓ ↓
EC2 Server 1 EC2 Server 2
(Node.js) (Node.js)
